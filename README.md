##安装步骤
- 安装node.js项目API接口
```
git clone https://github.com/Binaryify/NeteaseCloudMusicApi
```
```
node app.js
```
- 安装此项目
```
git clone https://github.com/Binaryify/NeteaseCloudMusicApi
npm install
npm run serve
```
